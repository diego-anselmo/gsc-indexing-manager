GSC Indexing Manager

Verifique e solicite a indexação das suas páginas no Google de forma automática.

Seus dados ficam apenas no seu computador.

Configuração Inicial

Antes de começar, precisamos configurar sua conexão com o Google. Siga os passos abaixo:

1

Criar projeto no Google Cloud

Acesse o Google Cloud Console e crie um novo projeto (ou use um existente).

2

Ativar as APIs necessárias

No seu projeto, ative estas duas APIs:

  • Google Search Console API
  • Web Search Indexing API
3

Criar credenciais OAuth

Vá em Credenciais → Criar credenciais → ID do cliente OAuth

  • Tipo: App para computador (Desktop)
  • Nome: qualquer nome (ex: "GSC Indexing")
  • Adicione o URI de redirecionamento: http://localhost:5000/auth/callback

Após criar, clique em Fazer download do JSON.

4

Tela de consentimento OAuth

Configure a Tela de consentimento:

  • Tipo: Externo
  • Em "Usuários de teste", adicione seu email
5

Envie o arquivo JSON aqui

Arraste o arquivo JSON aqui

ou clique para selecionar

Iniciar Verificação

Vamos verificar o status de indexação de todas as URLs dos sitemaps selecionados e solicitar a indexação das que não estão no Google. Os sitemaps selecionados também serão enviados automaticamente para ajudar o Googlebot a descobrir as páginas.

Inspecionar URL Individual

Verifique uma URL específica para saber se ela pode ser rastreada e indexada pelo Google. O site deve estar selecionado no Passo 1.
